# TypeScript Boolean

摘要：在本教程中，你将学习 TypeScript `boolean` 数据类型。

TypeScript 布尔类型允许两个值：`true`和`false`。它是 TypeScript 的原始类型之一。例如：



```ts
let pending: boolean;
pending = true;
// after a while
// ..
pending = false;
```

JavaScript 有`Boolean`类型，它引用非原始的封装对象。`Boolean`类型的`B`是大写的，这与`boolean`类型不同。

最好避免使用`Boolean`类型。
